This is a Read Me for Anakin's Lightsaber!

Left or A:  Swipe Left
Right or D: Swipe Right
Up or W:    Saber Thrust
Space:      Turn on/off Saber
Mouse:      Spins Saber on X Axis

XML File: anakinSaber.xml

Command to run after compilation: 
"Game anakinSaber.xml"

You'll have to change the Makefile and build_game.bat paths to match your own!
